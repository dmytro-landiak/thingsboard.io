Trendz Analytics in ThingsBoard Cloud is a fully managed, scalable, and fault-tolerant version of the Trendz Analytics add-on. It is already integrated into ThingsBoard Cloud and can be used to analyze your data immediately.

**Benefits:**

- **Improved time to market.** Save time on maintaining the platform or configuring its features.
- **Reduced costs.** Infrastructure costs are shared between platform users.
- **High availability.** Trendz Analytics uses a microservices architecture deployed across multiple availability zones.
- **Data durability.** The platform uses data replication and backup procedures to ensure you don't lose data.

## Prerequisites

You need an active ThingsBoard Cloud account. If you don't have one yet, <a href="https://thingsboard.cloud/signup" target="_blank">sign up here</a>.

## Activate Trendz Analytics

Trendz Analytics is already connected to the ThingsBoard Cloud infrastructure — the only step required is to activate it.

See the <DocLink product={Products.TRENDZ} path="activation/public-cloud">Cloud (US / EU) Activation</DocLink> guide for instructions.
