## v1.12.0 (Dec 31, 2024)

**Improvements:**

- Added prediction model training and validation framework.
- Added background tasks for prediction models.
- Added prediction fields support in calculated fields and views.
- Added ability to persist forecasts as telemetry in ThingsBoard.
- Added custom translation and locales management.
- Added support for new ThingsBoard widgets (ThingsBoard 3.9+).
- Added support for two-factor authentication.
- Added ability to filter data by date fields inside views.
